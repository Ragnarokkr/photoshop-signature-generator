# Signature Generator for Photoshop

> Make one signature, repeat it for all your documents.

(Really) Simple script to make a text signature and replicate it on every document currently opened into your Photoshop session.

It's possible to define the text, font, size, and position into the document.

**Tested only with Photoshop CS6**
